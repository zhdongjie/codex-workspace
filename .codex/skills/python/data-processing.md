# name
data-processing

# goal
编写或修改用于数据清洗、转换或市场分析的 Python 脚本

# steps
1. 明确数据的输入源（如 CSV、数据库、API）和期望的输出格式
2. 编写数据加载逻辑，并进行必要的空值或异常值处理
3. 实现核心的数据转换或统计算法
4. 添加清晰的日志输出，展示数据处理的进度和关键指标
5. 使用 filesystem 写入脚本代码

# constraints
- **绝对安全**：对原始输入数据必须是只读操作 (Read-only)，禁止在处理过程中覆盖或销毁原始数据源
- 在处理大规模历史数据时，优先考虑向量化操作或批处理，避免低效的 for 循环
- 代码中涉及的各种阈值、时间窗口参数必须提取为可配置项
